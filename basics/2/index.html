ID space

Every Kademlia node and key is identified by a 160-bit identifier.

For simplicity, we will be looking at a 6-bit identifier space for this visualization.

The 6-bit identifier space can be represented by a complete binary tree with 26 leaves. Each leaf represents a unique ID based on the "0" or "1" bit on each edge from the root to the leaf.
